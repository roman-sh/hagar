Privacy Policy for the Rexail Catalog Sync Extension

This document outlines the privacy practices for the "Rexail Catalog Sync" Chrome extension, an internal business tool provided by Teivah Solutions.

Data Collection and Usage

This extension is designed for a single purpose: to synchronize a product catalog from the Rexail backoffice to the Hagar inventory management system. To achieve this, the extension accesses two specific pieces of data:

  • Authentication Token: The extension accesses a temporary authentication token ("tarfash") from the active Rexail webpage. This token is used exclusively to make an authenticated API call to retrieve the product catalog. The token is used in-memory for this single request and is not stored, logged, or shared for any other purpose.
  • Website Content (Store Name): The extension reads the store's name from the HTML content of the active backoffice webpage. This store name is sent to the Hagar backend along with the catalog data for the sole purpose of identifying the correct store for the sync.

Data Storage and Sharing

No personal or user data is stored by the extension itself. The authentication token and store name are used ephemerally and are not saved after the sync operation is complete. We do not sell or transfer any user data to third parties.
